# Datos necesarios para personalizar la demo

## Negocio

- Nombre exacto de la peluqueria/barberia.
- Direccion completa.
- Telefono o WhatsApp.
- Instagram, Facebook o TikTok si aplica.
- Logo, si existe.
- Colores preferidos o estilo visual.

## Servicios

Por cada servicio:

- nombre;
- descripcion corta;
- precio;
- duracion aproximada;
- si requiere anticipo o pago completo.

## Agenda

- dias de atencion;
- hora de apertura;
- hora de cierre;
- descansos o dias no laborales;
- si se permitiran reservas el mismo dia;
- tiempo minimo para cancelar o cambiar cita.

## Pagos

- QR de Nequi, Daviplata, Bancolombia u otro medio;
- nombre del titular;
- si se cobra anticipo;
- porcentaje o valor del anticipo;
- como se confirma manualmente el pago.

## Ubicacion

- direccion exacta para mapa;
- latitud y longitud, si ya las tienes;
- barrio o referencia para clientes.

## Publicacion

- `DATABASE_URL` de Supabase;
- URL de Vercel cuando se cree;
- confirmar si el backend seguira en Render;
- dominio deseado, si se va a comprar.

## Material para presentacion

- fotos reales del local;
- fotos de cortes/trabajos;
- nombre del peluquero o equipo;
- politicas basicas de reserva y cancelacion.
